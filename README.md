# HealthGuard

A healthcare transparency platform with five integrated modules for public health analysis.

## Modules

- **PriceVision** - Hospital price transparency and compliance analytics
- **DrugWatch** - Drug pricing comparison (US vs international)
- **FoodScore** - Food health scoring with NOVA classification
- **RuralAccess** - Healthcare shortage mapping and HPSA analysis
- **ChronicCare** - Chronic disease prediction and intervention prioritization

## Tech Stack

- Flask backend with Jinja2 templates
- PyTorch ML models for predictions
- Bootstrap 5 + custom CSS design system
- Leaflet.js for interactive maps
- Chart.js for data visualization
